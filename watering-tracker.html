Протокол полива

Версия 1.0 | Последнее обновление: 23.01.2025

1. Параметры раствора по группам

Группа Тип растений PPM (CalMag) Universal pH
A Чувствительные 80 1.5 мл/л 5.8–6.3
B Ароидные 120 6 мл/л 6.0–6.5
C Неприхотливые 150 6 мл/л 6.2–6.7

2. Справочник растений по группам

A — Чувствительные (PPM 80)
  • Калатея Calathea — полосатые листья
  • Спатифиллум Spathiphyllum — белые цветы
  • Гузмания Guzmania — красная розетка
  • Алоказия Alocasia — стрелки-листья
  • Антуриум Anthurium — глянцевые красные
  • Папоротник Nephrolepis — перистые листья
B — Ароидные (PPM 120)
  • Монстера Monstera — дырчатые листья
  • Филодендрон Philodendron — сердцевидные
  • Эпипремнум Epipremnum — лиана, пятнистые
  • Аглаонема Aglaonema — пёстрые листья
  • Бегония Begonia — в горошек
  • Хамедорея Chamaedorea — пальма
  • Хлорофитум Chlorophytum — паучок
C — Неприхотливые (PPM 150)
  • Драцена Dracaena — пальма с полосками
  • Кротон Codiaeum — яркие разноцветные
  • Сансевиерия Sansevieria — тёщин язык
Хищные — ТОЛЬКО чистый RO
  • Дионея Dionaea — мухоловка (PPM < 10, БЕЗ добавок!)

3. Добавки

DYNOMYCO Spark (микориза) 1 ч.л. (~3 г) / 4 л
ATAMI CalMag до целевого PPM

4. Порядок приготовления

Стандартный полив (удобрение)
  1. Набрать RO воду (PPM ≈ 0)
  2. Добавить CalMag до целевого PPM группы
  3. Добавить DYNOMYCO — 1 ч.л. на 4 л, размешать
  4. Добавить Universal — доза по группе
  5. Подождать 2–3 минуты
  6. Измерить pH, скорректировать при необходимости
  7. Полить растения
Промывка (каждый 4-й полив)
  1. Набрать RO воду
  2. Добавить CalMag до целевого PPM группы
  3. Измерить pH, скорректировать
  4. Полить растения — БЕЗ удобрений
