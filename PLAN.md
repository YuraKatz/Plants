## 1. Проблемы растений — Диагностика и решения

### Структура страницы `plant-problems.html`
- [ ] Создать страницу с визуальным гидом по проблемам
- [ ] Интерактивный поиск по симптомам

### Симптомы листьев
- [ ] **Желтеют листья**
  - Нижние листья → естественное старение
  - Все листья → перелив / недостаток света / дефицит азота
  - Между жилками → хлороз (дефицит железа/магния)
  - Пятнами → солнечный ожог / грибок

- [ ] **Сохнут кончики листьев**
  - Коричневые сухие → низкая влажность воздуха
  - Коричневые мягкие → перелив
  - Только кончики → соли в воде / перекорм

- [ ] **Листья скручиваются**
  - Вниз → перелив
  - Вверх → недолив / жара
  - Внутрь → вредители (клещ)

- [ ] **Пятна на листьях**
  - Коричневые сухие → ожог / грибок
  - Коричневые мокрые → бактериальная инфекция
  - Белый налёт → мучнистая роса
  - Чёрные точки → сажистый грибок

- [ ] **Листья опадают**
  - Резко много → стресс (переезд, сквозняк, перепад температур)
  - Постепенно снизу → нормально / недостаток света
  - Молодые листья → проблемы с корнями

### Проблемы роста
- [ ] **Не растёт**
  - Зимой → норма (период покоя)
  - Летом → мало света / тесный горшок / нет питания

- [ ] **Вытягивается**
  - Длинные междоузлия → недостаток света
  - Бледные листья → срочно добавить свет

- [ ] **Мелкие листья**
  - Недостаток света
  - Недостаток питания
  - Нужна пересадка

### Проблемы корней
- [ ] **Гниль корней**
  - Симптомы: вялые листья при мокром грунте, запах
  - Решение: вынуть, обрезать гниль, обработать, пересадить в сухой грунт

- [ ] **Корни вылезают**
  - Снизу из дренажа → пора пересаживать
  - Сверху → норма для некоторых (монстера)

## 2. Сезонный уход

### Структура страницы `seasonal-care.html`

### Общие правила по сезонам

#### Весна (Март-Май)
- [ ] Полив: постепенно увеличивать
- [ ] Удобрения: начинать подкормки (1/2 дозы → полная)
- [ ] Пересадка: лучшее время!
- [ ] Размножение: идеальный период
- [ ] Свет: можно выносить на балкон (постепенно)

#### Лето (Июнь-Август)
- [ ] Полив: максимальный, следить за пересыханием
- [ ] Удобрения: полная доза каждые 2 недели
- [ ] Опрыскивание: чаще (жара)
- [ ] Свет: защита от прямого солнца в полдень
- [ ] Проветривание: регулярное, но без сквозняков

#### Осень (Сентябрь-Ноябрь)
- [ ] Полив: постепенно сокращать
- [ ] Удобрения: уменьшить до 1/2 дозы → прекратить
- [ ] Пересадка: только срочная
- [ ] Занести растения с балкона до холодов
- [ ] Проверить на вредителей перед заносом!

#### Зима (Декабрь-Февраль)
- [ ] Полив: минимальный, с хорошей просушкой
- [ ] Удобрения: НЕ удобрять (исключение: цветущие)
- [ ] Свет: досветка для тропических
- [ ] Температура: беречь от батарей и холодных окон
- [ ] Влажность: увлажнитель обязателен

### Таблица по растениям

- [ ] Особые случаи:
  - Дионея: зимовка 0-10°C обязательна
  - Орхидеи: перепад температур для цветения
  - Кактусы/суккуленты: сухая холодная зимовка

### Гузмания
- [ ] Интересные факты:
  - Цветёт один раз в жизни, потом даёт деток и умирает
  - Эпифит — в природе растёт на деревьях
  - Название в честь испанского ботаника Анастасио Гусмана
- [ ] Происхождение: тропические леса Центральной и Южной Америки
- [ ] Частые проблемы:
  - Гниёт основание → вода попала в розетку + холод
  - Кончики сохнут → низкая влажность
  - Не цветёт → мало света, нужен этилен (яблоко в пакете)
- [ ] Токсичность: нетоксична
- [ ] Очищает воздух: да

### Драцена маргината
- [ ] Интересные факты:
  - "Драконово дерево" — сок краснеет на воздухе
  - Живёт до 10+ лет в квартире
  - NASA включило в топ очистителей воздуха
- [ ] Происхождение: Мадагаскар
- [ ] Частые проблемы:
  - Сохнут кончики → низкая влажность / фтор в воде
  - Опадают нижние листья → норма, формируется ствол
  - Мягкий ствол → перелив, гниль
- [ ] Токсичность: токсична для кошек и собак!
- [ ] Очищает воздух: да (формальдегид, бензол)

### Драцена фрагранс

- [ ] Токсичность: ядовит! Осторожно с детьми и питомцами
